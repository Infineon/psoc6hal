Looks up the resource block that connects to the specified pins from the provided resource pin mapping table. This is a convinience utility for cyhal_utils_get_resource() if the mappings is an array of known size.

Parameters
[in]pinThe pin to lookup the hardware block for
[in]mappingsThe mappings of pin to hardware block
Returns
The entry for the specified pin if it exists, or null if it doesn't.

typedef uint16_t cyhal_gpio_mapping_cfg_t

Bitfield representing the configuration of a GPIO (hsiom selection and mode). Use the CY_GPIO_CFG_GET_MODE and CY_GPIO_CFG_GET_HSIOM to extract the individual field values.
