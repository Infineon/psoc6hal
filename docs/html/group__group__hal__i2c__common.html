◆ cyhal_i2c_init()

cy_rslt_t cyhal_i2c_init ( cyhal_i2c_t *  obj,
cyhal_gpio_t  sda,
cyhal_gpio_t  scl,
const cyhal_clock_divider_t *  clk 
)

Initialize the I2C peripheral, and configures its specifieds pins. By default it is setup as a Master running at 400kHz. This can be changed by calling cyhal_i2c_configure(). NOTE: Master/Slave specific functions only work when the block is configured to be in that mode.

Parameters
[out]objThe I2C object
[in]sdaThe sda pin
[in]sclThe scl pin
[in]clkThe clock to use can be shared, if not provided a new clock will be allocated
Returns
The status of the init request
