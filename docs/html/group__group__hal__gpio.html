GPIO (General Purpose Input Output)
HAL Drivers

General Description

High level interface for interacting with the Cypress GPIO.

This driver provides configuration of GPIO pins. It also allows values to be read from and written to a pin directly from firmware.

If a GPIO is to be used as an input to or an output from a fixed function peripheral on the device, this driver should not be used directly. The driver for the fixed function peripheral will perform the necessary function calls to configure its GPIO pins.
